# 工程优化学习笔记

> **阅读说明：** 本仓库以 GitHub 页面为主要阅读环境，仅保证公式在 GitHub 上正常渲染；由于不同本地 Markdown 阅读器对数学公式的支持存在差异，本地预览效果不作保证。

本仓库不包含课程课本、课件、考试题等原始材料，只作为工程优化一般性知识点的学习整理。
